    Pregunta 42: Feature Contribution Explanation¶

    Respuesta:¶

    # prediction.py
def _generate_explanation(self, X_row) -> Dict[str, float]:  # Explicaci√≥n simple.
    # Enfoque simplificado: distancia a media por feature
    contributions = {}
    for col in self.feature_names:
        mean_val = self.feature_means.get(col, 0)  # Media de training.
        diff = (X_row[col] - mean_val) / (self.feature_stds.get(col, 1) + 1e-8)  # Z-score.

    Pregunta 73: Error Handling Philosophy¶

    ¬øCu√°l es tu filosof√≠a de manejo de errores?

    Respuesta:¶

    1. Fail fast: Validar inputs al inicio
    2. Errores espec√≠ficos: ValueError vs FileNotFoundError vs gen√©rico
    3. Logs contextuales: Incluir datos relevantes en el error
    4. Graceful degradation: Servicio funciona con capacidad reducida
    # Malo
try:
    do_something()
except Exception:                                # Captura todo: oculta bugs.
    pass                                         # Silencia errores: peligroso.

# Bueno
try:
    do_something(input_data)
except ValueError as e:                          # Captura espec√≠fica.
    logger.error(f"Invalid input {input_data}: {e}")  # Log con contexto.
    raise HTTPException(400, f"Invalid input: {e}")   # HTTP apropiado.
except FileNotFoundError as e:
    logger.error(f"Model file missing: {e}")
    raise HTTPException(503, "Model not available")   # 503: servicio no disponible.
